🔒 TERMOS DE USO

1. Natureza do Software

O sistema é uma aplicação ágil que opera exclusivamente de forma local no seu computador. Isso garante maior velocidade e independência de conexões externas contínuas durante o processamento.

Todos os dados inseridos são lidos em tempo real e descartados imediatamente quando a aba é fechada ou o arquivo é excluído da tela.

2. Responsabilidade do Utilizador

É de extrema responsabilidade do utilizador garantir que as etiquetas geradas pertencem aos pacientes corretos. A triagem e a conferência dos tubos devem sempre respeitar os padrões e protocolos exigidos.

3. Isenção de Garantias

O Software é fornecido como uma ferramenta de apoio para maximizar a produtividade diária. O utilizador deve sempre realizar a conferência visual das informações extraídas do documento antes de prosseguir com a impressão.

🛡️ POLÍTICA DE PRIVACIDADE

1. Princípio Fundamental

O sistema foi idealizado e desenvolvido com o princípio inegociável de privacidade e sigilo total dos dados sensíveis:

  • Nenhum dado é coletado, armazenado em banco de dados ou transmitido remotamente para servidores externos.
  • Tudo o que o sistema lê do PDF permanece apenas na memória viva e temporária do seu próprio dispositivo.

2. Segurança dos Dados

Como o processamento é 100% local (no seu navegador), a segurança das informações médicas depende diretamente da proteção e das boas práticas de uso do seu computador e rede interna.

📋 Uso Diário

Conferência

Confirme se os nomes lidos batem com os agendamentos. Se faltou alguém, use o botão "Novo Paciente".

Medição Exata

Use uma régua para medir os espaços da sua folha de etiquetas e insira na Configuração de Impressão (⚙️).

🖨️ Impressão Perfeita

Escala 100% (CRÍTICO)

Na hora de imprimir, a "Escala" do PDF deve estar em "Padrão" ou "100%". Nunca use "Ajustar à página", ou as etiquetas sairão desalinhadas.

Qualidade

Mude as propriedades da impressora para qualidade "Alta" para garantir a leitura nítida das datas e nomes.

🖥️ Telas e Monitores

Espaço

O sistema foi feito para computadores. Se a tela de configuração de folhas ficar espremida ou cortada, pressione F11 para usar o modo Tela Cheia.
